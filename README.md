# didox-cleaner-privacy
Privacy Policy — DidoX Cleaner

DidoX Cleaner does not collect, store, transmit, or share any personal or sensitive user data.

The extension works only on https://didox.uz/ and is used solely to hide popup and advertisement elements for a cleaner browsing experience.

No user information is collected, sold, or transferred to third parties.

The extension does not use remote code, analytics, tracking systems, or external servers.
